**Unmute — INTELLIGENT SIGN LANGUAGE INTERPRETATION SYSTEM**

Unmute is an AI-powered web platform that enables real-time sign language translation and interactive learning. It combines computer vision, machine learning, and gamification to improve communication and make sign language learning engaging.

**🚀 Features**
**🔤 Communication**
Real-time sign → text translation using computer vision
Text → sign language video generation
Emotion detection for enhanced understanding
**🎮 Gamified Learning**
Earn XP and level up
Daily streak tracking
Badge collection system
Structured learning path (Day-based progression)
**🔐 User System**
Firebase Authentication (Google + Email login)
Persistent user progress (Firestore)
Personalized learning experience
**🧠 How It Works**
Camera captures hand gestures
Backend ML model predicts sign meaning
System converts gestures into text
Emotion model analyzes facial expression
Learning system tracks progress and rewards users

**🏗 Tech Stack**
**Frontend**

**Backend**
